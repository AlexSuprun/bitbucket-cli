## Quick Start

```bash
# Install the CLI
bun install -g bitbucket-cli

# Authenticate with Bitbucket
bb auth login -u your-username -p your-app-password

# Clone a repository
bb repo clone myworkspace/myrepo

# List open pull requests
bb pr list
```
